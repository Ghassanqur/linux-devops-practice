# Linux Basic Commands

## Navigation
pwd
ls
ls -la
cd /home
cd ..

## File Operations
touch file.txt
mkdir test
rm file.txt
rm -r test
cp file.txt copy.txt
mv file.txt newfile.txt

## File Viewing
cat file.txt
less file.txt
head file.txt
tail file.txt

## Permissions
chmod 755 file.sh
chmod +x script.sh
chown user:user file.txt

## System Info
whoami
uname -a
top
df -h
free -m
