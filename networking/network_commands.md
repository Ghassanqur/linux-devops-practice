# Linux Networking Commands

## Check IP Address
ip a
hostname -I

## Check Connectivity
ping google.com

## DNS Lookup
nslookup google.com
dig google.com

## Check Open Ports
netstat -tulnp
ss -tulnp

## Test HTTP Requests
curl http://example.com
wget http://example.com/file

## Routing Info
ip route

## Check Hostname
hostname
